The DIGITAL_FORENSICS folder contains a collection of Python scripts designed to assist with Digital Forensics and Incident Response (DFIR) investigations. These tools are intended to help analyze digital evidence, investigate suspicious activity, and support forensic analysis workflows.

The scripts in this directory focus on common forensic tasks such as:

File integrity verification using cryptographic hashing

Log analysis for detecting suspicious events

Extraction of metadata from files and images

Investigation of browser artifacts and user activity

Basic malware investigation using hash comparison
